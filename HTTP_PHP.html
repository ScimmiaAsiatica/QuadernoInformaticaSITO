Protocollo HTTP e linguaggio PHP


HTTP

HTTP (Hyper Text Transfer Protocol) e' un protocollo che opera al settimo livello del modello ISO/OSI, nel livello di applicazione. Permette ai client
(browser) e ai server di scambiarsi informazioni. Il protocollo HTTP per comunicare utilizza sessioni dove ciascuna sessione inizia stabilendo per prima cosa
una connessione TCP al server, effettuando poi una richiesta (request) contenente un URL. In risposta (response) il server produce e restituisce il file richiesto
e, nella situazione piu' semplice, chiude la connessine TCP immediatamente. HTTP e' un protocollo senza memoria di stato (stateless): ne' il server ne' il client mantengono
informazioni relative ai messaggi precedentemente scambiati.



Pagine statiche e dinamiche

Le pagine che formano il Web possono essere:

  • STATICHE = quando esse presentano gli stessi contenuti a tutti gli utenti che vi accedono
  • DINAMICHE = quando possono mostrare contenuti diversi in relazione alle azioni effettuate dagli utenti stessi



PHP

PHP (Pre Hypertext Processor) e' un linguaggio che viene interpretato dal motore PHP (PHP engine), chiamato Zend. La sintassi prevede che l'inizio dello script
venga marcato dal tag di apertura "?PHP e da quello di chiusura ?". PHP e' un linguaggio case sensitive, di derivazione C, infatti prevede che le istruzioni terminino
sempre con il punto e virgola.
